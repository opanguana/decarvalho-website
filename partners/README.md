# Logos de Parceiros e Clientes

Esta pasta contém os logos exibidos no banner «Parceiros e Clientes» da homepage.

## Abordagem: fundo transparente (como o logo DeCarvalho)

Todos os logos devem ser **PNG com fundo transparente**, para se integrarem no fundo da página sem caixas brancas ou coloridas. Use a mesma abordagem do logo principal (`logos/logo-decarvalho-transparent.png`).

Se tiver ficheiros com fundo branco ou colorido:
- Abra em editor de imagem (GIMP, Photoshop, Photopea, etc.) ou use um serviço como [remove.bg](https://www.remove.bg) para remover o fundo.
- Exporte como **PNG com transparência**.
- Coloque os ficheiros nesta pasta com os nomes indicados em baixo.

## Ficheiros necessários

| Ficheiro | Organização |
|----------|-------------|
| `undp-logo.png` | UNDP (United Nations Development Programme) |
| `snv-logo.png` | SNV Netherlands Development Organisation |
| `oxfam-logo.png` | Oxfam |
| `tdh-logo.png` | Terre des hommes |
| `amodefa-logo.png` | AMODEFA |
| `actionaid-logo.png` | ActionAid |
| `forum-mulher-logo.png` | Fórum Mulher |

**Formato:** PNG, fundo transparente, altura sugerida ~48–80px para boa leitura no marquee.
