# Como adicionar os logos dos Parceiros e Clientes

Os logos que forneceu (UNDP, SNV, Oxfam, Terre des hommes, AMODEFA, ActionAid, Fórum Mulher) devem ficar nesta pasta com **fundo transparente**, como o logo DeCarvalho.

## Passos

1. **Remover o fundo** de cada imagem (branco, preto ou colorido), se ainda tiver:
   - Pode usar [remove.bg](https://www.remove.bg), GIMP, Photoshop ou Photopea.
   - Exporte como **PNG com transparência**.

2. **Copiar** os 7 ficheiros para a pasta `partners/` e **dar estes nomes**:
   - UNDP → `undp-logo.png`
   - SNV → `snv-logo.png`
   - Oxfam → `oxfam-logo.png`
   - Terre des hommes → `tdh-logo.png`
   - AMODEFA → `amodefa-logo.png`
   - ActionAid → `actionaid-logo.png`
   - Fórum Mulher → `forum-mulher-logo.png`

3. **Tamanho:** altura entre 48px e 80px é suficiente para o banner. Pode redimensionar no editor antes de guardar.

Depois de guardar os 7 PNG com transparência nesta pasta, o site mostrará automaticamente os novos logos na secção «Parceiros e Clientes».
