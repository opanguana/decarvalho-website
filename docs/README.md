## Apresentação PDF (Quem somos)

Coloque na pasta **`docs/`** o ficheiro da apresentação para o link no site funcionar:

- **apresentacao-empresa_final.pdf** — Apresentação da Empresa (versão final).  
  Se o seu ficheiro se chama `apresentação da Empresa _ final.pdf`, copie-o para esta pasta e renomeie para `apresentacao-empresa_final.pdf` (sem espaços nem acentos, para compatibilidade com todos os browsers).

O link «Descarregar apresentação da empresa (PDF)» na página **Quem somos** aponta para este ficheiro.
