# Documentação do Logo Baile da Saudade

**Baile da Saudade** — Convivência intergeracional, valorização cultural e bem-estar da 3.ª idade.  
Uma iniciativa DeCarvalho Consultorias & Serviços.

---

## 1. Ficheiros do projeto

Coloque nesta pasta os seguintes ficheiros:

| Ficheiro | Formato | Descrição |
|----------|---------|-----------|
| `logo-baile-saudade.png` | PNG | Logo para uso no site (recomendado: fundo transparente, aprox. 400–600px de largura) |
| `logo-baile-saudade.ai` | Adobe Illustrator | Projeto vetorial editável (Adobe) |
| `logo-baile-saudade.eps` | EPS | Formato vetorial editável (compatível com Illustrator, CorelDRAW, Inkscape) |
| `logo-baile-saudade.svg` | SVG | Formato vetorial editável (web, Inkscape, Figma) |

O site usa o ficheiro `logo-baile-saudade.png` da pasta `logos/` (raiz). Copie o PNG final para `logos/logo-baile-saudade.png` após exportar.

---

## 2. Paleta de cores oficial

| Nome | Hex | RGB | Uso |
|------|-----|-----|-----|
| **Baile (principal)** | `#B45309` | 180, 83, 9 | Cor principal do logo, elementos de destaque |
| **Baile Light** | `#D97706` | 217, 119, 6 | Gradientes, hover, variações claras |
| **Baile Soft** | `rgba(180, 83, 9, 0.12)` | — | Fundos suaves, destaques em baixa opacidade |

### Valores para CSS (site)
```css
--baile: #B45309;
--baile-light: #D97706;
--baile-soft: rgba(180, 83, 9, 0.12);
```

---

## 3. Especificações para exportação

- **PNG:** fundo transparente, resolução adequada para web (1x: 400–600px largura; 2x para retina: 800–1200px)
- **Adobe Illustrator:** guardar como `.ai` e/ou `.eps` (compatibility: CS6 ou superior)
- **SVG:** exportar para web, otimizado, sem estilos inline desnecessários

---

## 4. Onde o logo é usado no site

- Página principal (index): secção «As nossas soluções»
- Página Baile da Saudade: hero e secções
- Footer (em algumas variantes)
